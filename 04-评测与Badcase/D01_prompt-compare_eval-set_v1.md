# D01 Prompt Compare Eval Set v1

## 任务

评估 3 个 Prompt 版本在“岗位能力清单去重与优先级提炼”任务中的表现。

## 输入样本

- 固定输入文件：`../99-收件箱/inbox_2026-04-18_day1_jd_raw_input.md`

## 评测维度

| 维度 | 通过标准 |
| --- | --- |
| 去重能力 | 不直接照抄原始材料，能合并近义能力 |
| 结构化程度 | 输出能稳定落到固定结构，而不是散文式长答 |
| 优先级判断 | 能清晰给出 P0 / P1 / P2 或等价优先级 |
| 可执行性 | 能提出一周内可执行的任务，而不是泛泛建议 |
| 知识库适配性 | 结果可直接进入知识库，而不是必须二次重写 |

## 当前观察结果

| 版本 | 去重能力 | 结构化程度 | 优先级判断 | 可执行性 | 知识库适配性 | 备注 |
| --- | --- | --- | --- | --- | --- | --- |
| A | 中 | 低 | 中 | 中 | 低 | 更像“学习建议长答” |
| B | 高 | 中高 | 高 | 高 | 中高 | 结构已明显改善，但仍偏长 |
| C | 高 | 高 | 高 | 高 | 高 | 当前最佳，但仍存在截断问题 |

## 下一轮评测建议

1. 同一输入重复运行 3 次，检查一致性。
2. 增加第二份原始输入，验证迁移性。
3. 对最佳版本单独测试更紧的输出预算。
