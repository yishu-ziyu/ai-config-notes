这张表用来判断每天产出的文件里，哪些已经具备作品集价值，哪些还只是过程记录。

## 记录规则

每完成一个正式产物，就补一行。

字段解释：

- 日期：产出日期
- 能力模块：Prompt / Agent / Workflow / API / 需求分析 / Eval / Badcase / PRD

- 是否可投递：`是 / 否 / 待补证据`
- 还缺什么证据：截图、结果表、对比数据、badcase、复盘等
