# Day1 原始输入：AI产品岗位能力清单摘录

## 说明

这份材料不是完整 JD 汇总，而是从现有超长岗位能力清单里抽出的代表性片段，用作 Day 1 Prompt 对比实验的统一输入。

## 原始材料

- Agent 产品经理：Prompt Engineering、Agent 工作流编排、LLM API 调用、机器人产品 Sense、工程思维、多模态交互、Badcase 反馈机制。
- AI 产品经理：AI 产品设计能力、B 端产品经验、用户洞察与需求分析、数据分析能力、跨团队协作能力。
- AI Agent 产品经理：Agent 产品设计、大模型/NLP 原理、对话系统设计、竞品分析、用户需求理解。
- 大模型产品实习：AIGC/大语言模型理解、产品意识与逻辑思维、用户研究与数据分析、产品设计与交互基础、AI/编程基础知识。
- AI 产品实习生：生成式 AI 工具使用经验、Prompt 调优、PRD 撰写与原型设计、市场与用户研究、项目协调与推进。
- AI 工具产品实习：PRD 与原型输出、Prompt 优化、文档能力、AI 编程工具使用经验、产品审美能力。
- 平台型 AI 产品：产品规划能力、用户同理心、技术理解力、系统化思考能力、行业洞察与业务逻辑。
- AI 应用产品：效果评估与迭代、数据分析、Badcase 管理、Benchmark 设计、模型评测能力。
- 通用高频项：逻辑分析、需求拆解、方案设计、快速学习能力、自驱力、沟通表达、项目推进。
- 技术高频项：Coze、Dify、LangChain/Langflow、Prompt Engineering、RAG/Fine-tuning/Embedding、API/SDK、Python/JavaScript、vibe coding。
