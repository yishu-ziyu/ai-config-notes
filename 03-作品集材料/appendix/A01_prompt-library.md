## 当前可复用版本

### 1. 岗位能力提炼 Prompt

适用任务：
从一段岗位描述里提炼核心能力、优先级和学习重点。

```text
你现在扮演岗位分析师。

任务：
根据下面的岗位描述，提炼 5 个最核心能力，并按 P0 / P1 / P2 排优先级。

输出要求：
1. 用中文
2. 输出表格，字段为：能力 / 为什么重要 / 优先级
3. 不要照抄原文
4. 不要超过 200 字

材料：
[岗位描述]
```

留下它的原因：

- 任务清楚
- 输出清楚
- 能直接复用
- 不容易跑偏

### 2. 会议记录转 PRD Prompt

适用任务：
把一段讨论内容收束成能继续推进的产品文档。

```text
你现在扮演产品经理助理。

任务：
把下面的会议记录整理成一版 PRD 草稿。

输出结构固定为：
1. 背景
2. 目标
3. 用户问题
4. 功能方案
5. 待确认问题

要求：
- 用中文
- 不要补不存在的信息
- 不确定的地方标记“待确认”
```

留下它的原因：

- 结果清楚
- 风险可控
- 很适合真实工作场景
