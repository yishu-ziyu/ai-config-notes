# A01 Prompt Library

这里只放最后留下来的 Prompt。

不放失败草稿。
不放过程废话。
只放真正还能再用一次的版本。

## 当前候选

### 1. 岗位能力提炼 Prompt

适用任务：
从一段岗位描述里提炼核心能力、优先级和学习重点。

推荐结构：

```text
你现在扮演 AI 产品岗位分析师。

任务：
根据下面的岗位描述，提炼 5 个最核心能力，并按 P0 / P1 / P2 排优先级。

输出要求：
1. 用中文
2. 输出表格，字段为：能力 / 为什么重要 / 优先级
3. 不要照抄原文
4. 不要超过 200 字

材料：
[岗位描述]
```

为什么留下它：

- 任务清楚
- 输出清楚
- 能直接复用
- 不容易跑偏
