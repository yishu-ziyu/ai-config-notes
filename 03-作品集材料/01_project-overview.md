# Project Overview

## 冲刺主题

7 天内完成一轮 AI 产品岗位作品集冲刺，目标不是“多学一点”，而是沉淀可展示的 demo、case study 和评测材料。

## 计划证明的能力

1. Prompt 设计与效果优化
2. Agent / Workflow 方案设计
3. API 接入与快速原型
4. 用户研究与需求拆解
5. Eval / Badcase 迭代闭环
