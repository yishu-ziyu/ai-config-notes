# Gmail IMAP 配置

Gmail 设置截图（已移除）。

在这一步中选择保持默认选项就好，直接点 Save Changes：

| 选项 | 推荐 | 原因 |
|------|------|------|
| Auto-Expunge | Off | Hermes 处理邮件更稳妥，避免误删 |
| When expunged | Archive | 删除时自动归档，不彻底删除 |

点击 Save Changes 即可，然后去生成应用密码。

## 配置步骤

### 1. 开启 Gmail IMAP

1. 打开 Gmail → 设置（右上角齿轮）→ 查看所有设置
2. 切换到 **转发和 POP/IMAP** 标签页
3. 在 IMAP 访问部分选择 **启用 IMAP**
4. 保存更改

### 2. 获取应用密码（Gmail 必须用）

> Gmail 不允许用普通密码，必须用应用密码。

1. 访问 https://myaccount.google.com/apppasswords
2. 如果没有「应用密码」选项，需要先开启两步验证
3. 点击应用密码 → 选择应用（邮件）→ 选择设备（Mac/其他）
4. Google 会生成一个 16 位密码，复制保存好

### 3. Hermes 配置填写

> Gmail 应用密码截图已移除（包含敏感凭证）。

| 字段 | 值 |
|------|------|
| IMAP host | imap.gmail.com |
| IMAP port | 993 |
| Username | 你的 Gmail 地址（不含 @gmail.com） |
| Password | 应用密码（16位） |
